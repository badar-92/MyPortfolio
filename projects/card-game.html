Game Rules (Bhabhi/Thulla)

Core Rules

Follow Suit: Players must play the same suit as led if they have it
Tochoo/Thulla: If player cannot follow suit, they may play any card
Trick Resolution: Highest card of led suit wins if all follow suit
Tochoo Penalty: If someone cannot follow suit, highest led suit collects all cards
First Move: Game must start with Ace of Spades
Win Condition: First to empty hand wins. Last player loses (Bhabhi/Thulla)

Game Terminology

Bhabhi
The winner/loser title for the last remaining player with cards
Thulla
Alternative name for the game, referring to the penalty system
Tochoo
When a player cannot follow suit and plays a different suit
Trick
A round of cards where each player plays one card
Follow Suit
Mandatory rule: play same suit as first card in trick if possible
Suit Avoidance
CPU strategy to avoid picking up cards of same suit consecutively

Technical Implementation

Built with Python and Pygame using object-oriented design patterns for maintainable and extensible game architecture.

Class Architecture

Modular class structure for clean separation of concerns:

  • Game Class: Main game controller and state manager
  • Card Class: Card representation with suit and rank
  • Player Class: Player state and hand management
  • Button Class: Interactive UI elements
  • MovingCard Class: Animation controller for card movements
  • ScrollableHand Class: Hand interface with scrolling

Code Structure

class Card:
    def __init__(self, rank, suit):
        self.rank = rank
        self.suit = suit
        self.value = RANK_VALUE[rank]

    def draw(self, surf, x, y, face=True, disabled=False):
        # Draw card with dim overlay if disabled
        rect = pygame.Rect(x, y, CARD_W, CARD_H)
        pygame.draw.rect(surf, WHITE if face else DARK_GRAY, rect, border_radius=6)
        
        if face:
            color = RED if self.suit in ['♥', '♦'] else BLACK
            font = pygame.font.SysFont('arial', 20, bold=True)
            rsurf = font.render(self.rank, True, color)
            surf.blit(rsurf, (x+6, y+6))
            
        if disabled:  # Dim overlay for unplayable cards
            overlay = pygame.Surface((CARD_W, CARD_H), pygame.SRCALPHA)
            overlay.fill((80, 80, 80, 140))
            surf.blit(overlay, (x, y))
