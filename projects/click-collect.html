Click & Collect

Product showcase & local pickup platform – list, discover, and collect items seamlessly

Project Overview

Click & Collect is a full‑featured web application that enables users to browse products, submit their own items for sale, and arrange local pickup. Built as a modern alternative to classified platforms, it combines an intuitive public interface with a powerful admin panel for managing listings, categories, and customer inquiries.

Live Website: https://clickncollect.xo.je/clickcollect/public/

The Challenge

Local communities and small sellers often struggle with fragmented platforms that are either too complex (e‑commerce) or too limited (social media groups). Key issues included:

  • No centralized place to list items for local pickup
  • Manual product uploads and image handling
  • Lack of search/filter capabilities for nearby items
  • Difficulty for admins to moderate listings and messages
  • No structured category or tag system

The Solution

I built a clean, responsive platform with separate public and admin areas. The system empowers users to submit products with images, while administrators have full control over content, categories, and user messages.

  • Public product listing with grid view, search, and category filters
  • Simple submission form with image upload (drag & drop, preview)
  • Admin dashboard for product/category/message management
  • Contact page with message storage and admin notification
  • Bulk actions (export, delete) and activity logs
  • Secure authentication for admin users with session management

Key Features

Product Management

  • Submit products with title, description, price, and multiple images
  • Automatic image resizing and webp conversion for performance
  • Category assignment and custom tags
  • Soft delete and restore from admin

Discovery & Filters

  • Full‑text search across product titles and descriptions
  • Filter by category, price range, and date added
  • Lazy‑loaded product grid for fast browsing
  • Product detail page with contact seller option

Image Upload

  • Drag‑and‑drop or click to upload (multiple files)
  • Client‑side preview and validation
  • Secure server‑side handling (type/size checks)
  • Automatic thumbnail generation

Admin Panel

  • Role‑based login for administrators
  • CRUD for products, categories, and messages
  • Export all data to CSV/Excel
  • Activity logs and system health check

Contact & Messaging

  • Contact form sends email and stores inquiry in database
  • Admin can reply and mark messages as read
  • Spam protection with honeypot and rate limiting

Maintenance Mode

  • Toggle maintenance mode from admin settings
  • Custom message for public visitors
  • Whitelist IPs for admin access during maintenance

Technical Implementation

Built with a modular PHP architecture following best practices for security and maintainability.

PHP 8.0+ MySQL 8.0 JavaScript (ES6) Bootstrap 5 AJAX jQuery (limited) PDO Git Composer PHPMailer WebP conversion LazyLoad

Database Architecture

Relational schema with 10+ tables including:

  • products – core product data with image references
  • categories – hierarchical category structure
  • admins – secure admin credentials (bcrypt hashed)
  • messages – contact form submissions
  • logs – admin actions and system events
  • settings – maintenance mode, site title, etc.

Security & Performance

  • Prepared statements for all SQL queries (PDO)
  • XSS protection – output escaping with htmlspecialchars
  • CSRF tokens on all forms
  • File upload validation – mime type, size, and virus scan (ClamAV optional)
  • Session security – regeneration, HTTP-only cookies
  • Lazy loading images to improve initial page load
