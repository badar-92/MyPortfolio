MES Society Portal

Comprehensive University Society Management System for University of Lahore

Project Overview

The MES (Mechanical Engineering Society) Portal is a comprehensive web application designed to manage all aspects of a university society. Built for the University of Lahore, this system handles public website content, member management, event organization, competition management, and administrative controls through an intuitive role-based interface.

Live Website: https://mesuol.xo.je

The Challenge

University societies face numerous challenges in managing their operations:

  • Fragmented communication across multiple platforms
  • Manual membership application and approval processes
  • Difficulty in organizing and tracking events and competitions
  • Lack of centralized member database and ID management
  • Inefficient content management for public website
  • No proper role-based access control for different society positions

The Solution

I developed a full-stack web application with separate modules for public users, society members, and administrators. The system features:

  • Multi-role authentication system with 10+ different user roles
  • Public-facing website with dynamic content management
  • Member portal with personalized dashboards and digital ID cards
  • Event management system with registration and tracking
  • Competition platform with team registration and result management
  • Recruitment system with application tracking and interview scheduling

Technical Implementation

Built with modern web technologies following MVC architecture for clean code separation and maintainability.

PHP 8.0+ MySQL 8.0 JavaScript ES6+ Bootstrap 5 AJAX jQuery PDO RESTful APIs Git Composer PHPMailer QR Code Generator

Database Architecture

Complex relational database with 15+ tables including:

  • Users: Multi-role user management with permissions
  • Events: Complete event data with registration tracking
  • Competitions: Team-based competition management
  • Applications: Membership and event applications
  • Galleries: Image and document management
  • Notifications: Real-time user notifications

Security Features

  • SQL Injection Prevention: PDO prepared statements
  • XSS Protection: Input sanitization and output escaping
  • CSRF Protection: Token-based form validation
  • Password Security: bcrypt hashing with salt
  • Session Management: Secure session handling with regeneration
  • File Upload Security: Type validation and virus scanning
